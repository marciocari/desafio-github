Download do Git: https://git-scm.com/downloads
Git Basch: terminal para acessar e usar o Git.
Adicionar os novos objetos ao repositório: git add . OU git add -A
Estado (conferência): git status
Comentar antes de enviar para o repositório: git commit -m "Inclusão das anotações iniciais"
Enviar para o repositório (push): git push origin main
